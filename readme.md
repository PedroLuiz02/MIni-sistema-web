Este projeto é um sistema web desenvolvido em Python com o framework Flask, que reúne várias aplicações simples em uma única interface. A página principal funciona como um menu que direciona o usuário para diferentes funcionalidades, como soma de números, dados de petshop, teste de personalidade e monitoramento de contaminação.

O objetivo é demonstrar o conceitos de rotas, formulários, integração com banco de dados e front-end, servindo como um exercício prático de desenvolvimento web e organização modular em Flask.

Instalar flask e pandas via pip install, ou utilizar o requeriments.<br>
  -pip install flask pandas<br>
  -pip install -r requirements.txt

Inicializar no terminal com "python app.py"
